# AT86RF215-Breakout-Board
A 4-layer AT86RF215 Breakout Board made using KiCAD 8.0. It can be configured using the SPI bus to recieve and transmit simultaneously on the 2.4Ghz ISM and 900Mhz ISM bands. It is intended to be manufactured by JLCPCB.
